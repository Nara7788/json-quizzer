Field Explanations:

  • question (required): The question text
  • options (required): Array of answer choices
  • type (required): Either "single" (one correct answer) or "multiple" (multiple correct answers)
  • correct_answer (for single): Index number of the correct option (0-based)
  • correct_answers (for multiple): Array of index numbers for correct options (0-based)
  • tip (optional): Additional explanation shown after answering with dynamic option references

Important Notes:

  • Array indices start from 0 (first option = 0, second = 1, etc.)
  • For single-choice questions, use correct_answer with a number
  • For multiple-choice questions, use correct_answers with an array of numbers
  • In tips, use [[opt1]], [[opt2]], etc. to reference options dynamically
  • [[opt1]] refers to the 1st option in the array, [[opt2]] to the 2nd, and so on
  • Option references automatically adapt to shuffled order (e.g., if shuffled, [[opt1]] becomes the appropriate letter)
  • Tips are optional but recommended for better learning experience

Tip Format Examples:

  • Basic reference: "[[opt2]] is correct because..."
  • Multiple references: "[[opt1]] and [[opt3]] are correct. [[opt2]] is incorrect because..."
  • Mixed with letters: "Option [[opt1]] is the best choice, while [[opt4]] would cause issues."
